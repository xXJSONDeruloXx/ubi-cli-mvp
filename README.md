# ubi-cli-mvp

Research-driven MVP terminal CLI for Ubisoft Connect / Uplay account, library, product, and manifest inspection.

Status: bootstrap in progress. See `docs/research.md`, `docs/architecture.md`, and `docs/roadmap.md` for the source-backed plan.[1][2][4][5][6][9][11]

## Current bootstrap commands

```bash
npm install
npm run build
node dist/index.js doctor
node dist/index.js config show
```
